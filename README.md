# Rebootstrap

A little shell script that helps to install a minimal OS and then to boot on it.

Features:
* Supported distros:
  * Debian
  * Potentially any `debootstrap` supported distro
  * Potentially any `archstrap` supported distro (TODO)
* Grub:
  * mbr
  * efi (TODO)
* Migration:
  * `resolv.conf`
  * `/root/*`
  * `/homes/*`
  * `/systemd/networks/*`
